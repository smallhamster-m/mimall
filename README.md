# mimall
使用vue全家桶开发的防小米商城
技术栈：Vue、VueRouter、Vuex、Axios、ElemenUI
主要有登录、注册、商城首页、商城产品站、商城产品详情、购物车、订单确认、订单支付、订单列表等功能

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
